# ✈ 炸飞机 · BOOM!

> 一款基于 Web 的「炸飞机」棋盘小游戏，支持单人搜索、人机对战、双人对战三种模式。

---

## 🎮 游戏简介

「炸飞机」是一款经典的中国棋盘推理游戏。在一个 10×10 的网格中，隐藏着一架完整飞机，玩家需要通过「轰炸」格子来定位飞机的位置：

- 💣 **命中机身** → 提示"炸到了"
- 💥 **命中机头** → 直接获胜！
- 🏆 **炸出整架飞机** → 也算胜利

---

## 🕹️ 三种游戏模式

### 🎯 单人模式

- 15 次轰炸机会
- 在棋盘中搜索隐藏的飞机
- 右侧图例展示当前飞机的机型参考
- 命中机头或炸毁整架飞机即获胜
- 机会耗尽则失败

### 🤖 人机对战

- 你和 AI 各守一架飞机
- 轮流轰炸对方防区
- AI 拥有智能策略（见下方 AI 系统介绍）
- 左侧攻击 AI 防区，右侧展示你的防区被 AI 轰炸的结果
- 无次数限制，先摧毁对方飞机者获胜

### ⚔️ 双人对战

- 摇骰子决定先手
- 桌面端：双棋盘，每人各守一架飞机，轮流攻击对方防区
- 移动端：共享一块棋盘，轮流轰炸同一架飞机
- 无次数限制，先摧毁对方飞机者获胜

## ✈️ 8 种飞机机型

每种飞机有独特的形状和 4 个朝向（上/下/左/右），共 **32 种变体**：

| 机型 | 格子数 | 形状特征 |
|:---:|:---:|:---|
| 战斗机 | 8 | 窄机身，叉形尾翼 |
| 客机 | 11 | 宽大主翼（5格宽），经典轮廓 |
| 隐身机 | 10 | 三角翼布局 |
| 轰炸机 | 10 | 长机身，双层翼 |
| 超音速客机 | 8 | 后置三角翼，叉形尾 |
| 运输机 | 9 | 超宽翼展（7格宽），极短机身 |
| 鸭翼战机 | 10 | 前鸭翼 + 大后翼 |
| 双尾战机 | 11 | P-38 风格双尾撑 |

示意图示例（战斗机，朝上）：

```
  . ★ .    ← 机头（命中即胜）
  . ■ .    ← 机身
  ■ ■ ■    ← 主翼
  . ■ .    ← 机身
  ■ . ■    ← 叉形尾翼
```

## 🤖 AI 智能系统

人机对战模式中的 AI 采用了三层智能策略：

### 1. 概率搜索（Hunt Mode）

- 遍历所有 8 种飞机 × 4 个朝向 = **32 种模板**
- 对棋盘上每个空格子，计算"被多少种合法放置方案覆盖"
- 优先轰炸概率最高的格子
- 每次攻击后根据结果动态更新概率矩阵

### 2. 追踪模式（Track Mode）

- 一旦命中机身，立即将命中点的上下左右空邻居加入优先队列
- 按知识分数排序，优先攻击概率最高的邻居
- 在追踪和搜索之间智能切换

### 3. 知识更新（Knowledge Update）

- 每次命中后重新计算概率矩阵
- 排除与已有命中/未命中结果矛盾的飞机放置方案
- 搜索范围随游戏进行逐步缩小，精确度越来越高

### AI 决策流程

```
有未追踪的命中且邻居队列非空？
  ├─ 是 → 追踪模式：从邻居队列选知识分数最高的格子
  └─ 否 → 搜索模式：从全局空格子选知识分数最高的格子
```

## 🎨 界面设计

### 主题：Clear Sky — 晴空

采用明亮清新的亮色主题设计：

- **基底色**：晴空灰白 `#f0f4f8`
- **卡片**：纯白 `#ffffff` + 精细阴影层次
- **强调色**：天空蓝 `#3b82f6` / 珊瑚红 `#f97066`
- **命中反馈**：浅红渐变 `#fef2f2 → #fecaca`
- **机头命中**：暖黄渐变 `#fffbeb → #fde68a`
- **未命中**：浅灰 `#f1f5f9`

### 响应式设计

- 桌面端（>900px）：三栏横向布局，双棋盘并排
- 平板端（640-900px）：中等布局自适应
- 移动端（≤640px）：纵向堆叠布局，共享棋盘模式

### 交互体验

- 格子 hover 蓝色光环 + 缩放效果
- 命中脉冲动画 / 机头命中爆发动画 / 未命中闪烁
- 骰子摇动 + 落地弹性动画
- 胜利彩纸特效 + 音效反馈
- Web Audio API 原生音效（命中 / 未命中 / 机头 / 胜利）

## 🛠️ 技术架构

### 技术栈

- **纯前端**：HTML5 + CSS3 + Vanilla JavaScript
- **零依赖**：无任何框架或第三方库
- **部署**：Docker + Nginx

### 文件结构

```
plane-hunt/
├── index.html    # 页面结构（3个游戏屏幕 + 结果弹窗 + 音效开关）
├── style.css     # 样式（CSS变量主题 + 响应式 + 动画）
├── game.js       # 游戏逻辑（12个功能模块）
└── Dockerfile    # Docker 部署配置
```

### game.js 模块划分

| 模块 | 行号 | 功能 |
|:---|:---|:---|
| 1. Plane Templates | - | 8种飞机模板定义 + 旋转 + 随机生成 |
| 2. Board State | - | 棋盘状态管理（攻击/判定/揭示） |
| 3. Single Player | - | 单人模式（15次机会） |
| 4. Two Player Desktop | - | 双人桌面端（双棋盘） |
| 5. Two Player Mobile | - | 双人移动端（共享棋盘） |
| 6. Dice Logic | - | 摇骰子先手判定 |
| 7. Board Rendering | - | 棋盘/图例渲染 |
| 8. Sound Effects | - | Web Audio API 音效 |
| 9. Result Modal | - | 结果弹窗 + 彩纸特效 |
| 10. AI Engine | - | 智能AI（概率搜索 + 追踪 + 知识更新） |
| 11. AI Game Flow | - | 人机对战流程控制 |
| 12. Screen Navigation | - | 页面切换 + 路由 |

## 🚀 快速开始

### 本地开发

```bash
# 方式一：Python 简易服务器
python3 -m http.server 8080

# 方式二：Node.js 服务器
npx serve .

# 方式三：直接用浏览器打开
open index.html
```

然后访问 http://localhost:8080

### Docker 部署

```bash
# 构建镜像
docker build -t plane-hunt .

# 运行容器
docker run -d -p 8080:80 plane-hunt

# 访问
open http://localhost:8080
```
